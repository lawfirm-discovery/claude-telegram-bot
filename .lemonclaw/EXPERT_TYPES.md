# EXPERT_TYPES — 전문가 타입별 핵심 기능 가이드

리걸몬스터 ERP는 전문가 타입별로 사용 가능한 기능이 다르다.
사용자의 전문가 타입을 파악하고, 해당 타입에 맞는 기능만 안내한다.
타입을 모를 경우 "어떤 전문가 타입이세요?" 한 번 물어본다.

---

## 공통 기능 (모든 타입)
- **상담관리**: 상담 요청 접수·처리, 채팅 상담, 일정 관리
- **AI 문서 작성**: 타입별 맞춤 서식 자동 생성
- **수임계약**: 계약서 작성·서명·관리
- **LDrive**: 클라우드 문서 저장소
- **달력**: 일정·기한 관리
- **알림**: 업무 알림·리마인더
- **조직관리**: 구성원·사무소 정보 관리

---

## 변호사 (LAWYER)
**핵심 ERP 기능:**
- **사건관리**: 소송사건·상담사건 생성·관리 (가장 핵심)
- **e-filing**: 법원 전자소송 연동 (변호사 전용)
- **판례·법령 검색**: 판례DB 검색, 법령 조회
- **AI 법률 문서**: 소장·답변서·계약서·법률의견서 자동 생성
- **기업자문**: 자문 계약 및 의견서 작성
- **타임시트·청구서**: 시간 기반 수임료 관리

**주요 업무 흐름:**
상담접수 → 사건개시(소송/상담) → 문서작성 → e-filing 또는 수임계약 → 수임료 청구

**주요 상담 영역:** 민사, 형사, 기업, 가족, 노동, 세무, 부동산

**자문 타입:** 법률의견서, 계약서 검토, 컴플라이언스, 위험성 평가, 일반자문

## 세무사 (TAX_ACCOUNTANT)
**핵심 ERP 기능:**
- **세무상담 관리**: 상담 요청 접수·처리
- **AI 세무 문서**: 세무의견서·소득세/법인세 관련 서식 자동 생성
- **기업자문**: 세무 자문 계약 및 의견서 작성
- **수임계약**: 세무 수임 계약 관리

**주요 업무 흐름:**
세무상담 접수 → 자문 계약 → 세무 문서 작성 → 결과 보고 → 수임료 청구

**주요 상담 영역:** 소득세, 법인세, 부가가치세, 상속세, 증여세

**비해당 기능:** e-filing(법원전자소송), 소송사건, 판례검색

## 변리사 (PATENT_ATTORNEY)
**핵심 ERP 기능:**
- **특허상담 관리**: 특허·상표·디자인 상담 접수·처리
- **AI 특허 문서**: 출원서·의견서·거절이유서 자동 생성
- **출원 일정 관리**: 특허청 기한·OA(Office Action) 관리
- **수임계약**: 특허 수임 계약 관리

**주요 업무 흐름:**
상담 접수 → 선행기술 조사 → 출원 의뢰 → 문서 작성 → 특허청 제출 → 수임료 청구

**주요 상담 영역:** 특허, 상표, 디자인, 실용신안, 의장

**비해당 기능:** e-filing(법원), 소송사건(민사·형사), 판례검색(제한적)

상담 접수 → 서류 확인 → 등기 신청서 작성 → 법원/등기소 제출 → 수임료 청구

**주요 상담 영역:** 등기, 신탁, 상속, 부동산, 기업

**비해당 기능:** e-filing(소송), 소송사건(형사·민사 대리), 기업자문
