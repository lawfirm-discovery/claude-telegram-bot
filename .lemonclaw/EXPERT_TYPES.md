# EXPERT_TYPES — 사용자 타입별 ERP 기능 가이드

리걸몬스터 ERP는 사용자 타입(dtype)별로 접근 가능한 메뉴와 기능이 다르다.
사용자의 타입을 파악하고, 해당 타입에 맞는 기능을 안내한다.
타입을 모를 경우 "어떤 사용자 타입이세요? (일반/기업/변호사/세무사 등)" 한 번 물어본다.

---

## 사용자 타입 분류

### 일반 사용자 (비전문가)
- **일반 사용자** (INDIVIDUAL) — 개인 사용자
- **기업 사용자** (COMPANY) — 기업 법무팀/관리자
- **로스쿨 재학생** — 승인제

### 전문가 (11종)
변호사, 세무사, 변리사, 법무사, 회계사, 관세사, 감정평가사, 행정사, 노무사, 공증인, 속기사

---

## 공통 기능 (모든 사용자 타입)

다음 기능은 기업 사용자 포함 모든 사용자가 접근 가능:
- **대시보드**: ERP 홈
- **내 업무**: 일정관리, 할 일 목록, 메시지, 내 메모
- **전문가 상담**: 전문가에게 채팅 상담 요청
- **상담요청 관리**: 받은/보낸 상담 요청 목록
- **개인 LDrive**: 클라우드 문서 저장소
- **전자 계약**: 대시보드, 템플릿, 계약 관리, 비용 관리
- **법령 검색**: 법령 조회
- **판례 검색**: 판례DB 검색
- **법률 언론기사**: 법률 뉴스 검색
- **국회회의록**: 국회 회의록 검색
- **달력/알림**: 일정·기한·알림 관리

## 일반 사용자 (INDIVIDUAL)

일반 사용자도 소송사건을 직접 만들고 관리할 수 있다.

**핵심 기능:**
- **셀프 소송**: 변호사 없이 직접 소송 진행 (신규 등록 → 사건 목록 → 이력 조회)
- **위임 소송**: 전문가에게 소송 위임 (사건 참여 → 사건 목록)
- **AI 문서 작성**: 소장·답변서·계약서 등 법률 문서 자동 생성
- **문서 관리**: 작성 문서, 자주 쓰는 문서
- **전문가 상담**: 변호사 등 전문가에게 상담 요청

**주요 업무 흐름:**
법률 문제 발생 → 판례/법령 검색 → 셀프소송 등록 또는 전문가 상담 → AI 문서 작성 → 소송 진행 또는 위임

**비해당:** 기업법무 메뉴, e-filing(관리자전용), 레몬가디언(관리자전용)

---

## 기업 사용자 (COMPANY)

기업 사용자는 셀프소송/위임소송 대신 기업 법무 전용 메뉴를 사용한다.

**핵심 기능:**
- **기업 법무 대시보드**: 법무 현황 종합 뷰
- **법률 자문**: 외부 전문가에게 법률 자문 의뢰
- **계약 관리**: 기업 계약 관리 (전자계약 포함)
- **소송/분쟁 관리**: 기업 관련 소송·분쟁 관리
- **법률 문서함**: 기업 법률 문서 보관
- **컴플라이언스**: 규정 준수 관리
- **지식재산권 관리**: 기업 IP 포트폴리오 관리

**주요 업무 흐름:**
법무 이슈 발생 → 법률 자문 의뢰 → 계약/소송 관리 → 컴플라이언스 체크 → 문서 보관

**비해당:** 셀프소송, 위임소송, 문서관리(개인용)

## 로스쿨 재학생

승인제로 운영. 학습 목적 기능 중심.

**핵심 기능:**
- **판례·법령 검색**: 학습용 판례/법령 조회
- **AI 문서 작성**: 법률 문서 작성 연습
- **전문가 상담**: 멘토 변호사 상담

---

## 변호사 (LAWYER)

가장 많은 ERP 기능을 사용하는 핵심 타입.

**핵심 기능:**
- **사건관리**: 소송사건·상담사건 생성·관리 (전문가 전용 사건관리)
- **상담 접수·처리**: 일반/기업 사용자로부터 상담 요청 수락·관리
- **AI 법률 문서**: 소장·답변서·계약서·법률의견서 자동 생성
- **기업자문**: 기업 법률 자문 계약 및 의견서 작성
- **타임시트·청구서**: 시간 기반 수임료 관리

**주요 업무 흐름:**
상담 요청 접수 → 사건 개시(소송/상담) → AI 문서 작성 → 수임계약 → 수임료 청구

**상담 영역:** 민사, 형사, 기업, 가족, 노동, 세무, 부동산
**자문 타입:** 법률의견서, 계약서 검토, 컴플라이언스, 위험성 평가

- **기업자문**: 세무 자문 계약 및 의견서 작성
- **수임계약**: 세무 수임 계약 관리

**주요 업무 흐름:**
세무상담 접수 → 자문 계약 → 세무 문서 작성 → 결과 보고 → 수임료 청구

**상담 영역:** 소득세, 법인세, 부가가치세, 상속세, 증여세
