# Heartbeat Checklist

하트비트 시 아래 항목을 확인하세요. 문제가 있으면 즉시 텔레그램으로 알려주세요.

- [ ] Spring 서비스 정상 동작 확인 (localhost:8080 응답 여부)
- [ ] Frontend 서비스 정상 동작 확인 (localhost:3001 응답 여부)
- [ ] FastAPI 서비스 정상 동작 확인 (localhost:8001 응답 여부)
- [ ] 디스크 사용량 90% 미만인지 확인
- [ ] 메모리 사용량 95% 미만인지 확인

모든 항목 정상이면 "HEARTBEAT_OK"만 응답하세요.
문제 발견 시 어떤 항목이 비정상인지 구체적으로 알려주세요.
